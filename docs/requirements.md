# ResourceHub Requirements

## Project Overview
ResourceHub is a web application used to manage shared company
resources such as meeting rooms and vehicles.

The system prevents double bookings and allows employees
to reserve resources for specific time slots.

---

## Functional Requirements

• View resources  
• Create bookings  
• Prevent booking conflicts  
• Manage resources (CRUD)  

---

## Non-Functional Requirements

• Simple user interface  
• Fast response times  
• Error handling
